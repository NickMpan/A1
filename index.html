A Helping Hand

Overview

There is nothing worse than purchasing something that you have been so excited to get only to find that you can't figure out how to configure the settings properly. A helping hand is a great way for those who have a bit of free time and some knowledge in a certain area to pass that on to other people around the world. By teaching someone to do a specific task you are rewarded through your app and after learning something new you then give feedback on the tutorial based on how helpful it was and whether it has accurately addressed the request.

Motivation

As a 23 year old there ares still a lot of things that I have never gotten around to learning or have no one who can teach me. The reason I believe this app could be incredibly helpful to most people is because there is a large target audience of people who could use it, it can be incredibly appealing to those who have grown up with no role model to teach them certain things that they have wanted to learn. On top of that it allows some more specific tasks to be taugh to you by those who have taken the time to learn these themselves.

Description

The app will request a signup the same way any other social media platform would. When signing up, the user can register any qualifications that they have such as a Bachelor in IT or a Certificate in Carpentry which can be verified by the user to make any advice that is given more credible. Keep in mind that a degree or qualification is not required to make comments however people will be more likely to take advice from a qualified person. After the user creates an account can then make a post requesting help from other people in a completing a task. The task can be anything from "what is the best way to clean this stain from my shirt?" to "How can I train my dog to stop urinating in the house?". After asking a question the original poster will receive notifications for any comments received on their post and can hopefully take the next step to rectifying the issues that they face.

After completing registration with the app, the user can select multiple categories that they have knowledge on and will in turn have posts sorted based on their relevant qualifications and then displayed on their feed. When a user runs out of posts to be seen they can then look through other general requests to see if they can help in any other way. When a user is planning on giving advice to a poster, they can choose to post either step-by-step instructions as a comment or they can post a video to show a demonstration of how to do this specific task. The benefit to posting a video response is that it makes it easier to show how to perform this task. Say for example I would like to know how to take my bike chain off for cleaning and maintenance. I would be able to post a request with the hopes of someone posting a video showing the specific tools required and how to use the tools to take the bike chain off.

In addition to any advice already given, the original poster can then contact the commenter through direct message and ask any questions about the advice given with hopes of getting more information on the instructions. The direct message feature allows for the users to elaborate on advice, ask questions and also organise another method of contact such as a video call to have face to face instructions. When a user comments advice on a post, the original poster can then review this advice and give a one to five star rating based on how helpful the advice was. This rating is then averaged and displayed as an overall rating on the users profile for others to view when determining how trustworthy the advice is.

Tools and Technologies

The app will run like most other social media services do. It will run on a set of servers that will store all information and allow people to access it based on privacy settings. Any videos posted by a user in response to a comment will need to be stored on the servers for use by anyone who views the original post. Due to having certain confidential information, the servers will need to be incredibly secure and all passwords, email addresses and other sensitive information entered by a user will need to be automatically censored.

Skills Required

When designing the app there are many online builders that could be used to build the social media site however assuming we are building the website from scratch ourselves, we would need to use both HTML and CSS to personalise the website and do the basic design. Knowledge in server side programming would also be required, therefore use of a server side programming language such as PHP would be necessary to ensure that all privacy and security requirements are met to ensure that the website database is secure against any breaches.

Outcome

The overall hopes for this social media platform is that there is a large number of users across the world to help people in different places. When progressing with this app, advertising can be used to increase revenue as a main source of income to ensure that the app can be consistently maintained to ensure security standards are upheld. If a large enough number of users is gained, the ideal outcome is for a reward to be given to users on a monthly basis for those who have a large amount of positive reviews and consistently helpful comments.
